## Updating project

1. git add .

2. git commit -m "name"

3. git push origin main

## Creating new branch

1. git checkout -b "name"

2. git commit -m "name"

3. git push origin {branch}

---
### Checking out from branch and merging 

1. git checkout {branch-to-go-back}

2. git merge {branch-name}

3. git push origin main
